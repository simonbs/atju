localizedStringWithTable:

Returns a localized string associated to the receiver by the given table, returning the receiver if the string cannot be found. This method searches the main bundle before falling back on AirshipResources, allowing for developers to override or supplement any officially bundled localizations.

- (NSString *)localizedStringWithTable:(NSString *)table

Parameters

table

The table.

Return Value

The localized string corresponding to the key and table, or the key if it cannot be found.

localizedStringWithTable:fallbackLocale:

Returns a localized string associated to the receiver by the given table, falling back on the provided locale and finally the receiver if the string cannot be found. This method searches the main bundle before falling back on AirshipResources, allowing for developers to override or supplement any officially bundled localizations.
